## 🇯🇵 Japanese 50 Scripts Card Game (日文50音卡牌配對遊戲)

- 以同子音爲一組，最多 10 張卡片為一組
- 點擊翻開兩張相同讀音的卡牌會消失
- 卡片全部消失時會跳至下一組
- 點擊左下按鈕會回到首頁
